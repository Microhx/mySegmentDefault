### 项目介绍
  高仿项目来源于<a href='https://segmentfault.com/'>segmentfault app</a><br/>
  使用接口获取数据，使用<a href='http://www.cnblogs.com/lonelywolfmoutain/p/4233438.html'>jangod-template</a>注入到webview页面中,
  然后展示相应的数据。
